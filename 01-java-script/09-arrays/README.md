# Aula 09 - Arrays

## O que aprendemos?

Nesta aula, aprendemos:

- O que são arrays e qual seu propósito em JavaScript.
- A utilizar laços com arrays: `for`, `for...of` e `forEach`.
- Diferentes tipos de laço `for` aplicados a arrays.
- A trabalhar com métodos funcionais de array (como `map`, `filter`, `reduce`).
- A lidar com arrays contendo tipos de dados variados.
- Os conceitos de `Set` e `Map` como estruturas alternativas ao array.
- A manipular arrays na prática (adicionar, remover, transformar elementos).

## Pontos de atenção

- `forEach` não permite `break`/`continue` nem `return` para interromper a iteração — para isso, usar `for` tradicional ou métodos como `some`/`every`.
- Métodos funcionais (`map`, `filter`, `reduce`) não alteram o array original — retornam um novo array, o que ajuda a evitar efeitos colaterais.
- `Set` armazena apenas valores únicos (sem duplicatas); `Map` armazena pares chave-valor, com chaves de qualquer tipo (diferente de objetos comuns, que só aceitam strings/symbols como chave).
