# Aula 04 - Variáveis e Tipos de Dados

## O que aprendemos?

Nesta aula, aprendemos:

- A criar e modificar variáveis em JavaScript.
- O uso de `var`, `let` e `const` para declaração de variáveis, ressaltando suas diferenças e usos recomendados.
- O conceito de hoisting e seu impacto nas variáveis `var`, `let` e `const`.
- A identificar e diferenciar tipos primitivos de dados utilizando `typeof`.
- A diferença entre `undefined` e `null`.
- O uso de `symbol` para criação de valores únicos.
- A importância de seguir boas práticas na nomenclatura de variáveis, como utilizar camelCase.
- A relevância de escrever código legível e significativo, respeitando case sensitivity.
