## 🧠 Principais conceitos abordados

- **Lógica de programação**: algoritmos, fluxogramas, pensamento computacional.
- **Ambiente**: instalação e uso do Node.js, execução de scripts via terminal.
- **Fundamentos da linguagem**: variáveis (`var`, `let`, `const`), tipos primitivos, hoisting.
- **Operadores**: aritméticos, atribuição, comparação (`==` vs `===`) e lógicos.
- **Controle de fluxo**: `if/else`, operador ternário, `switch`, valores truthy/falsy.
- **Laços de repetição**: `for`, `while`, `do...while`, variantes do `for` (`for...of`, `for...in`).
- **Funções**: declaração, arrow functions, callbacks, HOF (Higher-Order Functions).
- **Estruturas de dados**: arrays, métodos funcionais (`map`, `filter`, `reduce`), objetos, `Set` e `Map`.
- **JavaScript moderno (ES6+)**: destructuring, spread/rest, módulos (`import`/`export`), conversão de tipos.
- **Introdução a tópicos avançados**: assincronismo (Event Loop, Promises, async/await) e orientação a objetos (`new`, classes).
