# Aula 03 - Configurando o Ambiente de Desenvolvimento

## O que aprendemos?

Nesta aula, aprendemos:

- Como instalar o Node.js em diferentes sistemas operacionais a partir do site oficial.
- A importância de verificar a instalação do Node.js usando `node -v`.
- Criar uma pasta de projeto e importá-la para o VS Code.
- Utilizar o terminal integrado do VS Code para criar e executar scripts JavaScript com Node.js.
- Compreender a diferença entre a execução de JavaScript no Node.js e no navegador.
- O funcionamento do Node.js como plataforma para executar JavaScript fora do navegador.
- O papel do motor V8 na interpretação de JavaScript no Node.js.
- A expansão das possibilidades do JavaScript no ambiente de servidor com Node.js.
