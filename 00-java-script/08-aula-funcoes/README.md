# Aula 08 - Funções

## O que aprendemos?

Nesta aula, aprendemos:

- O que são funções e qual seu propósito na organização do código.
- Como criar e executar funções em JavaScript.
- O funcionamento do `return` em funções.
- A sintaxe das arrow functions.
- As diferenças entre arrow function e function declaration/expression.
- O conceito de callbacks e HOF (Higher-Order Functions).
- A utilizar funções prontas (built-in) da linguagem.
- A capturar entrada de dados pelo terminal.
- A entender e utilizar o `require` para importar módulos.
- A construir um desafio prático: criação de um quiz utilizando funções.
- A refinar funções e parâmetros com apoio de IA.

## Pontos de atenção

- Arrow functions não possuem `this` próprio — herdam o `this` do escopo léxico onde foram definidas, diferente de `function` tradicional.
- HOF (Higher-Order Functions) são funções que recebem outra função como argumento e/ou retornam uma função (ex: `map`, `filter`, `reduce`).
- Função sem `return` explícito retorna `undefined` por padrão.
