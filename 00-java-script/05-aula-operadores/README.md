# Aula 05 - Operadores Aritméticos, de Comparação e Lógicos

## O que aprendemos?

Nesta aula, aprendemos:

- Como realizar operações matemáticas básicas e utilizar operadores aritméticos em JavaScript.
- A aplicar operadores de atribuição combinados e o operador de resto `%`.
- A utilizar operadores de incremento, decremento e de potência.
- A usar operadores de comparação e a diferença entre igualdade simples e estrita.
- A combinar condições lógicas com operadores "E", "OU" e "NÃO".
- A implementar comparações em cenários práticos utilizando `console.log`.
- A utilizar lógica para calcular troco em transações no JavaScript.
- A formatar a saída de resultados no terminal para exibir informações de forma clara.

## Pontos de atenção

- `==` faz coerção de tipo antes de comparar; `===` compara valor e tipo sem conversão.
  Preferir sempre `===` para evitar comparações inesperadas (ex: `"5" == 5` é `true`, mas `"5" === 5` é `false`).
