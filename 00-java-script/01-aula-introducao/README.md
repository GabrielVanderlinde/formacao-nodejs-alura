# Aula 01 - Primeiros Passos

## O que aprendemos?

Nesta aula, aprendemos:

- A definição e propósito de programar e algoritmos.
- A evolução e funções das linguagens de programação.
- Os conceitos de linguagens de baixo e alto nível.
- O paradigma da programação orientada a objetos.
- Novos paradigmas como Low-Code, No-Code e programação com AI.
- A função da linguagem de programação como intermediária e o papel dos compiladores.
- A história, evolução e importância do JavaScript, incluindo ECMAScript e frameworks.
- O uso do console do navegador e comandos JavaScript básicos como `console.log` e `alert`.
